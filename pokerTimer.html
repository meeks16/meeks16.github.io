Poker Timer Mobile App

Poker Tournament Timer App keeps track of the small and big blinds, counts down from a given time per round provide by the user, and alerts the players when the time is up. A new round is then created and the time automatically resets to back to the the given time. Once a new round starts, the small and big blinds are doubled. This is repeated until we have a winner or decide to end the game.

Features:

  1. Set the desired time per round
  2. Set desired blinds for each game
  3. Pause the timer for break time
  4. Manually Reset the time back up
  5. Adjust time while timer is running or not
  6. Adjust the blinds while timer is running or not
